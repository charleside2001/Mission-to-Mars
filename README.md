# Overview of The Mission To Mars Analysis

In this project,  `BeautifulSoup` and `Splinter`  was used to scrape full-resolution images of Marsâ€™s hemispheres and the titles. The scraped data was stored  on a `Mongo database` and `html` web application was used to display the data. The design of the web app was also altered using `CSS stylesheet` to accommodate these images.
